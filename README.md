# Правила допуска и сдачи экзаменов

### Допуск
— Сданы все обязательные домашние задания

### Зачёт
— При получении допуска автоматически ставится зачёт (оценка в БаРС будет Е)
— Зачёт не дает право прохода на второй семестр

### Экзамены
— Будет два экзамена: по вёрстке (21 января) и по JavaScript (23 января)
— Билет на экзамене состоит из теории и практики
— Экзаменатор ставит два балла: за теорию и за практику
— Итоговый балл — среднее с округлением вверх
— Исключение: если студент получил два по теории или практике — экзамен не сдан

### Автоматы
— Студенты, набравшие 70 баллов за домашки и доп.активности, не сдают практику
— Студенты, набравшие 15 баллов за квизы, не сдают теорию

Несамостоятельные выполнения заданий экзамена влекут за собой удаление с экзамена.
